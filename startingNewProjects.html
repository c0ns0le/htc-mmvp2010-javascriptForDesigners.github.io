Starting a New Project

The following instructions make the assumptions:

  • you have installed your command line tools on your machine.
  • you have setup your github.com account
  • you have installed the code editor Atom (min. version 1.18)

GitHub

login into your github account

click on Start a project

enter the Repository name

fill in the description

check the Initialize with a README

click on Create Repository

click on Settings tab

scroll down to GitHub pages

click on None and select Master branch

click Save

scroll back down to GitHub pages

right-click on ready to publish URL and copy the link address

click on Code tab (back at the top)

click on Edit button (right side of screen)

right-click in website field and paste GitHub pages URL

click Save button

click greenClone or download button

click the clipboard icon to copy the project URL to the clipboard

Atom

under View pull-down menu select Toggle Command Palette

in the command palette, type git clone and hit the return or enter key

in the Clone From field paste the project URL (copied from github)

update the To Directory with your preferred location

Update and/or create files

create any new files and save them to the project folder

update and save existing files in the project folder

under the Packages pull-down menu select GitHub

click the Stage All button in the upper right corner of the Git panel

enter your Commit message in the available field

click the commit button

click the Up Arrow button at the bottom of the Git panel

click the Push button

enter your GitHub login information if prompted to do so

GitHub

navigate back to your project on GitHub

click on the GitHub Pages link at the top of the project to see your webpages
